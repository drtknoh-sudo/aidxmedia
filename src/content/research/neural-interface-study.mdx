## Abstract

뇌-컴퓨터 인터페이스(BCI)는 신경 질환 환자의 의사소통과 운동 기능 회복에 혁명적인 가능성을 제시한다. 본 연구에서는 딥러닝 기반의 새로운 신호 처리 기법을 통해 비침습적 BCI의 한계를 극복했다.

## Introduction

비침습적 BCI는 두피에서 측정하는 EEG 신호를 기반으로 하기 때문에 다음과 같은 제약이 있다:

- **낮은 공간 해상도**: 두개골과 두피에 의한 신호 감쇠
- **높은 잡음 수준**: 근육 움직임, 환경 전자기파 등
- **개인 간 변이**: 해부학적 차이로 인한 신호 패턴 차이

본 연구는 이러한 한계를 극복하기 위한 새로운 접근법을 제시한다.

## Methods

### 데이터 수집

50명의 건강한 피험자를 대상으로 256채널 고밀도 EEG를 측정했다. 각 피험자는 다음 과제를 수행했다:

1. 운동 상상 과제 (좌/우 손 움직임)
2. 언어 상상 과제 (단어 생성)
3. 감정 유발 과제 (시각 자극)

### 신호 처리 파이프라인

```python
class BrainSignalProcessor:
    def __init__(self, channels=256):
        self.encoder = SpatialAttentionEncoder(channels)
        self.decoder = TemporalDecoder()

    def process(self, eeg_signal):
        # 공간적 주의 기제를 통한 소스 분리
        source_signals = self.encoder(eeg_signal)
        # 시간적 패턴 디코딩
        return self.decoder(source_signals)
```

## Results

### 공간 해상도 비교

| 방법 | 해상도 (mm) | 분류 정확도 |
|------|------------|------------|
| 기존 EEG | 25-30 | 75% |
| 본 연구 | 5-6 | 94% |
| 침습적 ECoG | 3-4 | 98% |

### 실시간 성능

본 시스템은 **50ms 지연** 내에 뇌 신호를 해석할 수 있어 실시간 응용에 적합하다.

## Discussion

본 연구의 핵심 기여는 다음과 같다:

1. **소스 분리 정확도 향상**: 공간적 주의 메커니즘을 통해 신호 혼합 문제 해결
2. **개인화 적응**: 전이 학습을 통해 새로운 사용자에게 빠르게 적응
3. **실용적 구현**: 상용 EEG 장비로도 적용 가능한 알고리즘

## Conclusion

비침습적 BCI의 성능 향상은 임상 적용을 위한 중요한 진전이다. 향후 연구에서는 더 다양한 환자 집단을 대상으로 임상 시험을 진행할 예정이다.
